# LoSuk — 로스트아크 숙제 체크리스트

로스트아크 원정대의 일일/주간 숙제를 캐릭터별 카드로 관리하는 Windows 앱입니다.

## 다운로드

**[Releases](https://github.com/IngBin/LoSuk-releases/releases)에서 최신 버전을 받으세요.**
설치 과정 없이 압축을 풀고 `LoSuk.exe`를 실행하면 됩니다. 새 버전이 나오면 앱이 알아서 알려주고, 버튼 한 번으로 업데이트됩니다.

## 주요 기능

### 캐릭터 관리
- 캐릭터명 하나만 입력하면 **원정대 전체 자동 등록** (아이템레벨 · 전투력 포함)
- 닉네임 옆 클래스 아이콘 표시, 드래그로 카드 순서 변경, 안 하는 캐릭터는 숨기기

### 일일 숙제 (카오스 던전 · 가디언 토벌)
- 매일 06:00 자동 리셋
- **휴식게이지** 자동 관리 — 5칸 게이지 바로 표시
  - 숙제를 안 한 날마다 반칸씩 자동 적립 (앱을 안 켜도 누적)
  - 숙제 체크 시 1칸 이상이면 1칸 자동 소모 (반칸만 있으면 소모 안 됨)
  - 게이지 숫자를 직접 수정해서 인게임과 맞출 수 있음

### 주간 컨텐츠 (낙원 · 에브니 큐브 · 할의 모래시계)
- 입장권/열쇠/티켓 보유 개수 기록 (주간 리셋과 무관하게 유지)

### 주간 레이드
- 매주 수요일 06:00 자동 리셋
- 아이템레벨에 맞춰 **골드가 높은 레이드만 자동 표시** (주간 골드 한도 3개 기준)
- 난이도 선택, 더보기 비용 자동 차감
- 캐릭터별 · 원정대 전체 **주간 골드 합계** (일반/귀속 구분)

## 데이터

체크 상태는 내 PC(`%LOCALAPPDATA%\LoSuk\data.json`)에만 저장됩니다. 서버로 전송되지 않습니다.
