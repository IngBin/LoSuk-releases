# LoSuk — 로스트아크 숙제 체크리스트

로스트아크 원정대의 일일/주간 숙제를 캐릭터별 카드로 관리하고, 각종 시세·효율까지 한 곳에서 확인하는 Windows 앱입니다.

## 다운로드

**반드시 아래 릴리스 페이지에서 각자 직접 받으세요.**

👉 **https://github.com/IngBin/LoSuk-releases/releases/latest**

Assets 목록의 **`LoSuk.exe`** 를 클릭해 내려받고 실행하면 됩니다. 설치 과정은 없고, 별도 런타임 설치도 필요 없습니다. 새 버전이 나오면 앱이 알아서 알려주고, 버튼 한 번으로 업데이트됩니다.

> ⚠️ **카카오톡·디스코드·드라이브 등으로 exe를 주고받지 마세요.**
> 파일이 약 65MB로 커서 전송 중 잘리거나 차단되어 **"실행이 안 되는"** 경우가 많습니다. (디스코드 무료는 용량 제한으로 아예 전송도 안 됩니다.) 꼭 위 릴리스 링크에서 **각자** 받아 주세요.

### 실행이 안 되거나 백신·윈도우가 막을 때

서명되지 않은 개인 제작 앱이라 처음엔 Windows·백신이 경고할 수 있습니다. 아래 순서대로 해제하면 됩니다.

1. **다운로드가 막힐 때** — 브라우저 다운로드 목록에서 `⋯ → 유지`(위험한 파일 보관)를 선택.
2. **파일 차단 해제** — `LoSuk.exe` 우클릭 → **속성** → 하단 **[차단 해제]** 체크 → 확인.
3. **SmartScreen 파란 창**("Windows에서 PC를 보호했습니다") — **추가 정보** → **실행** 버튼.
4. **백신이 삭제·격리할 때** — Windows 보안 → 바이러스 및 위협 방지 → **보호 기록**에서 해당 항목 **복원 / 허용**. (타사 백신은 예외 목록에 추가)

### 그래도 안 될 때 확인

- **파일 크기 확인**: `LoSuk.exe` 속성 → 크기가 **약 65MB**여야 정상입니다. 몇 KB~MB로 아주 작으면 전송 중 손상된 것이니 릴리스에서 다시 받으세요.
- **실행 환경**: Windows 10(1607 이상) 또는 Windows 11, 64비트. (Windows 7/8.1은 지원하지 않습니다.)
- 위를 다 해도 안 되면, 더블클릭 시 증상(아무 반응 없음 / 에러창 문구 / 잠깐 떴다 꺼짐)을 알려주시면 도와드리겠습니다.

## 주요 기능

상단 탭으로 화면을 전환합니다 — **숙제 · 낙원 · 경매 · 더보기 · 피해량컷 · 마리샵**.

### 숙제 (메인)
- 캐릭터명 하나만 입력하면 **원정대 전체 자동 등록** (아이템레벨 · 전투력 · 클래스 아이콘)
- 카드를 창 폭에 맞춰 줄바꿈, 카드별로 접어서 헤더만 보기 가능, 드래그로 순서 변경
- **일일 숙제**: 카오스 던전/가디언 토벌 + 휴식게이지(자동 적립·소모, 직접 수정)
- **주간 레이드**: 아이템레벨에 맞춰 골드 높은 레이드 자동 표시, 난이도·더보기 반영, 캐릭터별·원정대 골드 합계
- 매일 06:00 / 매주 수요일 06:00 자동 리셋

### 낙원
- 원정대에서 원하는 캐릭터만 골라 천상/증명/지옥 관리
- 천상 입장권 지급 이벤트 기간 설정 시 매주 자동 +1

### 경매
- 분배금 계산 (인원별 추천 입찰가·손익, 클릭 복사)
- 유물 각인서 / 보석(4티어) 시세 조회
- 악세서리 옵션 검색 — 고대 4티어, 부위별 옵션 최대 3개 조합, 품질 80/90

### 더보기
- 보스별 더보기 효율표 — 재료 실시간 시세로 손익 환산, 관문 종합

### 피해량컷
- 보스 체력 기준 1인분 / 강투컷 / 잔혈컷 + 초당 딜량

### 마리샵
- 비밀상점 현재 로테이션 효율표, 크리스탈 시세 자동 반영·로테이션 자동 갱신

## API 키

- 앱에 기본 키가 내장되어 있어 바로 사용할 수 있습니다.
- 여러 명이 함께 쓰면 호출 한도(분당 100회)를 나눠 쓰게 되므로, 각자 **개인 API 키**를 발급받아 입력하는 것을 권장합니다. (숙제 탭의 "발급 방법" 버튼에 안내가 있습니다.)
- 상단에 남은 호출 횟수와 초기화까지 남은 시간이 표시됩니다.
- 호출 한도를 초과해도 잠시 후 자동으로 재시도하며, 오래 걸릴 때는 초기화까지 남은 시간을 안내합니다.

## 데이터

체크 상태·설정은 내 PC(`%LOCALAPPDATA%\LoSuk\data.json`)에만 저장됩니다. 서버로 전송되지 않습니다.
