# LoSuk 배포 저장소

로스트아크 숙제 체크리스트(LoSuk)의 업데이트 배포용 저장소입니다.
- `version.json` — 앱이 시작 시 확인하는 최신 버전 정보
- 실행 파일은 [Releases](https://github.com/IngBin/LoSuk-releases/releases)에서 받을 수 있습니다.

## 업데이트 내역

### [1.1.1]
- 업데이트 배너에 현재 버전과 최신 버전 표기
- 하단 상태바에 현재 버전 표시
- 버전 확인 시 CDN 캐시 우회 (업데이트 안내가 최대 5분 늦게 뜨던 문제 해결)

### [1.1.0]
- 자동 업데이트 기능 추가 (새 버전 안내 배너 + 원클릭 업데이트)
- 벨가르딘 레이드 등록 (2026-08-05 출시일에 자동 활성화, 전 난이도 골드 포함)
- 더보기 골드 표시 및 총 획득 골드에서 차감 (종막·지평의 성당·세르카)
- 일반/귀속 골드 구분 (1710 이하 5:5, 초과 전액 일반, 성당 100% 귀속)
- 주간 컨텐츠 추가 (낙원 천상/증명/지옥, 에브니 큐브, 할의 모래시계 — 입장권/열쇠/티켓 카운터)
- 전투력 표시, 드래그로 카드 순서 변경, 섹션 접기(상태 저장), 창 위치 기억
- 캐릭터 숨김 기억 (X로 지운 캐릭터는 다시 불러와도 추가 안 됨)
